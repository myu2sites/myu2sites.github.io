There are 3 game modes:

Game mode 1:  -  Play On A Single iDevice  -  Two players play using a single iDevice.
Game mode 2:  -  Play On Two iDevices Via Local Wifi  -  Two players play with individual iDevice via local WiFi.
Game mode 3:  -  Online Live Via Game Center  -  Two players play against each other live across Internet via Game Center.

Have fun playing Checkers using your giant TV to share with family and friend instead of small iDevice screen.
